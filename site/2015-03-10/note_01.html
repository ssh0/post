xmonadとxmobarの設定と言うのを書いていたのですが、いろいろ改善点を直しているうちに、追記では収まらなくなていたので、新たにエントリーを書くことにしました。

xmoadを含むタイリングウィンドウマネージャの魅力については、他の人も多く語っているので、少しググってみると、その魅力に惹かれる人はいるんじゃないかなと思います。今回はxmonadとはなんぞや、といった話はせずに、今xmonadを使っている人向けに設定ファイルを晒す程度のものです。

また、当人はhaskellの知識はなく、stackoverflowやメーリングリストの情報から設定ファイルを書いているのであしからず。

keysToRemove x =
    [
        -- Unused gmrun binding

main :: IO ()

main = do

myManageHookShift = composeAll
            -- if you want to know className, type "$ xprop|grep CLASS" on shell
            [ className =? "Firefox"       --> mydoShift "  browser  "

            [ className =? "Gimp"             --> doFloat,

wsPP = xmobarPP { ppOrder           = \(ws:l:t:_)  -> [ws,t]

                , ppUrgent          = xmobarColor  colorWhite    colorNormalbg
                , ppVisible         = xmobarColor  colorWhite    colorNormalbg
                , ppHidden          = xmobarColor  colorWhite    colorNormalbg

設定ファイルに書いているキーバインドは以下

キー やっていること
Ctrl + Alt + l スクリーンをロック
Super + Return 端末urxvtを起動
Super + c フォーカスのあるウィンドウを削除
Super + p アプリケーションランチャーdmenuを起動。オプションは色の設定
Ctrl + Alt + f 以前に作成した簡易検索バーwebsearch.pyを起動
音量アップボタン(0x1008ff13) 音量を上げてunity環境と同じ音でフィードバック
音量ダウンボタン(0x1008ff11) 音量を下げてunity環境と同じ音でフィードバック
ミュートボタン(0x1008ff12) 音量をミュート
輝度アップボタン(0x1008FF02) xbacklightを使って画面の明るさを上げる
輝度ダウンボタン(0x1008FF03) xbacklightを使って画面の明るさを下げる
Super + f 全画面表示にするかの切り替え
Super + l マスター領域の横幅を大きくする
Super + a レイアウトがResizableTallのとき、領域の縦幅を小さくする
Super + z レイアウトがResizableTallのとき、領域の縦幅を大きくする
Super + h マスター領域の横幅を小さくする
Super + Ctrl + 矢印キー フォーカスのあるウィンドウを動かす
Super + Ctrl + h マウスポインタを左に動かすxte-left.sh
Super + Ctrl + j マウスポインタを下に動かすxte-down.sh
Super + Ctrl + k マウスポインタを上に動かすxte-up.sh
Super + Ctrl + l マウスポインタを右に動かすxte-right.sh
Super + Ctrl + Return マウスの左クリックをエミュレートxte-click.sh
Super + Right 次(右)のワークスペースに移動する
Super + Left 前(左)のワークスペースに移動する
Super + Shift + Right フォーカスのあるウィンドウを次のワークスペースに移動
Super + Shift + Left フォーカスのあるウィンドウを前にワークスペースに移動
Super + j フォーカスを次のウィンドウに移す
Super + k フォーカスを前のウィンドウに移す
Super + Shift + j フォーカスのあるウィンドウを次のタイル配置に移す
Super + Shift + k フォーカスのあるウィンドウを前のタイル配置に移す
Super + m フォーカスをマスター領域のウィンドウに移す
Super + Shift + m フォーカスのあるウィンドウをマスター領域に移す
Super + w 次のスクリーンにフォーカスを移す
Ctrl + Alt + t コンポジットマネージャcomptonの有効化、無効化の切り替えtoggle_compton.sh
PrintScreen(0xff61) 画面全体のスクリーンショットを撮るscreenshot.sh
Shift + PrintScreen(0xff61) 領域を選択してスクリーンショットを撮るscreenshot_select.sh
再生・ポーズボタン(0x1008ff14) 音楽再生アプリケーションcmusの起動or再生／ポーズcplay.sh

myLayoutで指定した3つのレイアウトをSuper+Spaceで切り替えることができます。

myStartupHookで、xmonadの起動時に呼び出すプログラムを設定しています。

myManageHookShiftで、特定のウィンドウは指定したワークスペースで開くように設定します。

myManageHookFloatで、特定のウィンドウはフロート表示するように設定します。
